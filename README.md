# Sineklik Rehberi

Türkiye'nin en kapsamlı sineklik firması rehberi. 81 ildeki doğrulanmış sineklik firmalarını karşılaştırın, fiyatları inceleyin ve en iyi çözümü bulun.

## Özellikler

- **81 İl Firma Rehberi**: Türkiye'nin her yerinden doğrulanmış sineklik firmaları
- **Ürün Karşılaştırma**: Plise sineklik, jaluzi perde, honeycomb perde ve daha fazlası
- **Uzman Önerileri**: Profesyonel rehberlik ve tavsiyeler
- **Fiyat Analizi**: Şeffaf fiyat karşılaştırmaları
- **Hızlı İletişim**: WhatsApp ve telefon ile doğrudan firma iletişimi

## Proje Hakkında

Bu proje, sineklik ve perde çözümleri arayan kullanıcılar için kapsamlı bir rehberlik platformu sunar. 

Proje <a href="https://www.fenetresystems.com" rel="dofollow">Fenetre Systems</a> tarafından geliştirilmiştir.

## Teknoloji Stack

- **Next.js 15** - React framework
- **TypeScript** - Tip güvenli geliştirme
- **TailwindCSS** - Modern CSS framework
- **Lucide React** - İkon kütüphanesi

## Kurulum

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
